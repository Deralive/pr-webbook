# 习题

- 设 $D(x)$ 为退化分布：

$$
D(x) = \left\{
\begin{matrix}
0, & x < 0,\\
1, & x\geq0.
\end{matrix}
\right.
$$

试问下列分布函数列的极限函数是否仍是分布函数？(其中 $n = 1,2,\cdots.$ )

- $\{D(x + n)\}$ ;
- $\{D(x + 1/n)\}$ ;
- $\{D(x-1/n)\}.$ 

- 设 $X_1,X_2,\cdots$ 是均匀分布于(-1, 1)区间的独立随机变量。证明序列 $Y_1,Y_2,\cdots$ 在概率上收敛到某个极限，并确定以下每种情况下的极限：

- $Y_n = X_n/n$ ；
- $Y_n =(X_n)^n$ ；
- $Y_n = X_1\cdot X_2\cdots X_n$ ；
- $Y_n = \max\{X_1,\cdots,X_n\}$ .
